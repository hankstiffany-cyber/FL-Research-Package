## COORDINATE DATABASE SUMMARY

### By Format Type

**Decimal GPS (94 coordinates)**
- Standard lat/long format
- Directly mappable
- Examples: 41.954269, -101.210973

**DMS Format (65 coordinates)**
- Degrees/Minutes/Seconds
- Directly convertible
- Examples: 29°55'37.95"N 58°33'6.87"E

**MilOrb Encoded (64 coordinates)**
- Format: +XXX+YYY or -XXX-YYY
- Requires NodeSpaces decoding
- Examples: +234+515, -008-936
